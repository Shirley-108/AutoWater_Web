# AutoWater_Web
## 介紹
使用 AWS IoT 製作一個可即時接收感測器資訊的網頁，可以查看當下溫濕度與種植植物的土壤濕度數值，判斷是否需要自動澆水。
## 使用技術
1. AWS IoT(裝置通訊協定 MQTT)
1. node.js
1. Arduino IDE
